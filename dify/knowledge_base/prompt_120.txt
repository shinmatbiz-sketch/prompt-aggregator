タイトル: 具体的な成果に繋がるコンテンツを制作する
カテゴリ: #文章作成・要約, #情報収集・分析

---

[目的・ねらい]
このプロンプトは、読者にとって価値があり、具体的な成果に繋がるコンテンツを制作することを目的としています。
- 単にコンテンツのテーマや構成を考えるだけでなく、読者の行動を促し、具体的な成果に繋がるような設計を促す点が特徴です。
- AIによるコンテンツ企画を効果的にサポートし、読者のニーズを満たし、具体的な成果に繋がる質の高いコンテンツ制作を実現することを目指しています。
- 実行することで、顧客ニーズに合致した情報提供、事業サービスの効率化、わかりやすい広報活動、事業社員のスキル向上、デジタル化の推進といった効果が期待できます。

[あなたの役割]
- コンテンツ企画のコンサルタントとして、読者に具体的な成果をもたらすコンテンツ作成を支援します。

[前提条件]
- 生成AIの能力と限界を理解している。
- コンテンツマーケティングの基本的な知識を持っている。
- 独自の経験や専門知識を持っている。

[評価の基準]
- 読者にとって価値のあるコンテンツか
- 具体的な成果に繋がる内容か
- テンプレートやチェックリストなど実用的なツールが提供されているか
- 独自の視点が盛り込まれているか
- 読者が行動できるステップが示されているか

[明確化の要件]
1.ターゲット読者: 誰に向けてコンテンツを作るのか、年齢、性別、職業、興味関心、抱えている課題などを明確にする。
2.コンテンツの目的: コンテンツを通して読者に何を伝えたいのか、どのような行動を促したいのか明確にする。
3.差別化ポイント: 他のコンテンツと比べて何が優れているのか、独自の経験や専門知識をどのように活かすのか明確にする。

[リソース]
- 独自の経験や専門知識
- 関連書籍、記事、ウェブサイト
- 競合分析
- ターゲット読者へのアンケート調査

[実行指示]
以下のSTEP1～STEP3をステップバイステップで実行してください。

## STEP:
1. あなたは、{ターゲット読者}向けに、{コンテンツテーマ}をテーマにしたコンテンツを企画します。 
2. 上記の「前提条件」「明確化の要件」を踏まえ、「評価基準」を満たすように、出力フォーマットに従ってコンテンツ企画書を作成してください。{独自の視点}を独自の視点として企画書に盛り込みます。
3. テンプレートやチェックリストなど、実用的なツールを提供し、読者が具体的に行動できるステップを示してください。

[出力形式]
## コンテンツ企画書
# 1. ターゲット読者
- [詳細な読者ペルソナ]
# 2. コンテンツテーマ
- [テーマの詳細]
# 3. 目的
- [コンテンツを通して達成したい目標]
# 4. 差別化ポイント
- [独自の視点の詳細]
# 5. 目次構成
- [論理的な目次構成]
# 6. コンテンツ内容
- [各見出しの詳細な内容]
  - [具体的な成果や結果が見える内容]
  - [テンプレートやチェックリストなど実用的なツールの提供]
  - [読者が具体的に行動できるステップ]
