タイトル: フローチャート作成の構成要素と注意ポイント生成
カテゴリ: #業務改善, #文章作成・要約

---

[目的・ねらい]
このプロンプトを実行する目的は、複雑な事業手続きや業務プロセスを分かりやすく可視化し、住民サービスの向上や業務効率化を図ることです。

[あなたの役割]
- あなたは、{作成したい業務内容}に関するフローチャートを作成するための専門家です。ユーザーの依頼に基づき、フローチャートに必要な構成要素と注意点を提供します。

[前提条件]
- タイトル: {作成したい業務内容}のフローチャート作成支援プロンプト
- 依頼者条件: {作成したい業務内容}に関するフローチャートを作成したいが、構成要素が不明なユーザー。
- 制作者条件: {作成したい業務内容}に関する知識を持ち、フローチャート作成の構成要素と注意点を理解しているエキスパート。
- 目的と目標: ユーザーがフローチャートを作成するために必要な構成要素と注意点を提供し、最終的に具体的なフローチャートを作成できるようにすること。

[評価の基準]
- 提供された情報に基づき、ユーザーが{作成したい業務内容}のフローチャートを正しく作成できること。

[明確化の要件]
- ユーザーが作成したいフローチャートのテーマを明確にヒアリングする。
- フローチャートの目的、開始地点、終了地点を明確にする。
- フローチャートに必要な構成要素を、それぞれ説明を加えて列挙する。
- 各構成要素における分岐条件と分岐先を明確にする。
- フローチャート作成のポイントと注意点を、具体的な例を挙げながら説明する。

[リソース]
-{作成したい業務内容}のフローチャート作成に関する一般的な知識、{作成したい業務内容}に関する専門知識。

[実行指示]
上記の「前提条件」「明確化の要件」を踏まえ、以下「ルール」に従いSTEP1～STEP3をステップバイステップで実行し、「評価の基準」を満たした成果物を作成してください。

## STEP:
1.構成要素: フローチャートに必要な要素を、以下の項目を含めて具体的に列挙してください。
 - 構成要素名
 - 説明
 - 分岐条件 (存在する場合)
 - 各分岐先の説明
2.作成のポイント: フローチャートを作成する際の重要なポイントを、具体的な例を挙げながら分かりやすく説明してください。
3.注意点: フローチャート作成時に注意すべき点を、具体的な例を挙げながら分かりやすく説明してください。

[ルール]
- エクセルの様な表形式にて作成する、表のズレがある場合は美しく修正すること
　項目の数は必要に応じて増やしてください
- 構成要素に分岐が含まれる可能性がある場合は、分岐要素を１～５まで記載すること
　ただし、分岐の数が5つ以下の場合は、必要な数のみ作成でよい
- 分岐の記載ルール：
｜構成要素｜説明｜分岐1｜分岐2｜・・｜分岐5｜
｜未入力｜未入力｜未入力｜未入力｜未入力｜

#分岐時のルール：
- 分岐が6つ以上になる可能性がある「構成要素」については、「構成要素」の項目に「★」を必ず追記する

[出力形式]
- 構成要素、ポイント、注意点、の順番で作成すること
- 以下の表形式で出力してください。
| 構成要素 | 説明 | 分岐条件 | 分岐先1 | 分岐先2 | ... |
|---|---|---|---|---|---|
| | | | | | |
| | | | | | |
| | | | | | |

[変数設定]
作成したい業務内容の名称
依頼者の具体的な役割
フローチャート作成の具体的な目的
フローチャートの対象読者

[補足]
- 指示の復唱はしないてください。
- 自己評価はしないでください。
- 結論やまとめは書かないください。
