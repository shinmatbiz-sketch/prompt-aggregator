タイトル: 会議議事録作成のための要約
カテゴリ: #文章作成・要約

---

[目的・ねらい]
このプロンプトは、AIを使って、会議議事録を作成することを目的にしています。

[あなたの役割]
会議の参加者または記録者として、会議の内容を理解し、要約する役割を担います。

[前提条件]
- タイトル: 会議議事録作成AI
- 依頼者条件: 会議の内容を要約し、議事録を作成したい人
- 制作者条件: 会議の内容を理解し、簡潔にまとめることができる人
- 目的と目標: 会議の要点を明確にし、記録として残す

[評価の基準]
- 正確性: 会議の内容が正確に要約されている
- 簡潔性: 議事録が簡潔で読みやすい
- 完結性: 必要な情報がすべて含まれている
- フォーマット準拠: 指定された出力フォーマットに従っている

[明確化の要件]
- 会議の録音データや文字起こしデータを提供
- 議題、決定事項、議論のポイントなどを明確に指示
- 出力フォーマットを具体的に指定

[リソース]
- {会議メモ}
- 議題リスト
- 関連資料

[実行指示]
{会議メモ}から{会議名}の議事録を作成してください。
{開催日時}に行われたこの会議は、{出席者一覧}が出席し、{議題}について議論されました。
会議の主な議論を要約し、{決定事項}を明確に記述してください。
{出力フォーマット}に従って出力してください。

## 要約ルール：
- 情報を全て、網羅的かつ構造的に要約してください。
- 内容は絶対に省かないようにすること

[出力形式]
```
## 会議名: [会議名]
# 開催日時:
- 日付: [開催日]
- 時間: [開始時間] ～ [終了時間]

# 出席者一覧:
- [出席者1]
- [出席者2]
- [出席者3]
...

# 議題:
1. [議題1]
    - [議題1に関する議論の要約]
2. [議題2]
    - [議題2に関する議論の要約]
...

# 決定事項:
- [決定事項1]
- [決定事項2]
...

```

# スタイルとトーン
明瞭でフォーマルなトーン

[変数設定]
会議名
開催日時
出席者一覧
議題
決定事項
会議メモ

[補足]
- 指示の復唱はしないてください。
- 自己評価はしないでください。
- 結論やまとめは書かないください。
