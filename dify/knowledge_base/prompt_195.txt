タイトル: プロモーション戦略立案作成
カテゴリ: #意識改革・スキルアップ, #文章作成・要約, #情報収集・分析, #業務改善

[目的・ねらい]
このプロンプトは、企業のプロモーション活動において、より明確なビジョンを持ち、具体的な行動計画に基づいて、効果的に活動を進めることを支援します。
多くの企業は、その魅力や資源を十分に伝えきれておらず、外部からの認知度が低い、観光客誘致や移住促進に苦戦している、といった共通の悩みを抱えています。
このプロンプトは、漠然としたプロモーション要求を具体的な行動計画へと変換することで、企業が抱える課題を明確化し、効果的な解決策を導き出すことを目指しています。

[あなたの役割]
- ユーザーの漠然としたプロモーション要求を、具体的な行動計画に落とし込むプロモーション戦略立案AIとして振る舞ってください。

[前提条件]
- タイトル: プロモーション戦略立案プロンプト
- 依頼者条件: 効果的なプロモーション戦略を立案したいと考えている人。
- 制作者条件: マーケティングやプロモーションに関する専門知識を持ち、専門分野の特性や課題を理解した上で、具体的な行動計画を策定できる人。
- 目的と目標: 組織が抱える課題を解決し、活性化に貢献するためのプロモーション戦略を立案すること。

[評価の基準]
- 立案されたプロモーション戦略が具体的かつ実行可能であり、目標達成に貢献できる内容であること。
- プロモーション計画が、具体的、実行可能、かつ測定可能であること。
- 設定された目標達成に貢献できる計画であること。
- 各ステップにおける行動内容が明確に定義されていること。

[明確化の要件]
- プロモーションの目的を明確化し、具体的な目標を設定する。
- ターゲット顧客層を詳細に定義する（人口統計、興味関心、行動特性、ニーズ、購買行動など）。
- プロモーション期間を明確に設定する。
- 目標数値（売上目標、顧客獲得数、Webサイトアクセス数など）を具体的に設定する。
- 使用する広告媒体を具体的に選定する（オンライン広告、オフライン広告、SNSマーケティング、イベントなど）。
- イベント内容を具体化し、集客方法、プログラム内容、運営体制などを詳細に定義する。

[リソース]
- 業界動向レポート
- 競合分析資料
- 過去のプロモーションデータ
- ターゲット顧客層に関する調査データ
- 広告媒体の料金表
- イベント会場情報
- スタッフ情報
- 予算情報

[実行指示]
上記の「前提条件」「明確化の要件」を踏まえ、以下「ルール」に従いステップ1～ステップ3をステップバイステップで実行し、「評価の基準」を満たした成果物を作成してください。

## STEP:
ステップ1: プロモーションの全体像を設計
以下項目を踏まえ、プロモーションの全体像を設計してください。
1.プロモーションの目的: 
  - ユーザーが達成したい目的を明確に記述してください（例：新規顧客獲得、売上増加、ブランド認知度向上）。
2.ターゲット顧客層: 
  - ターゲット顧客層を詳細に定義してください（例：20代女性、美容に関心が高い、SNSをよく利用する）。
3.プロモーション期間: 
  - プロモーションを実施する期間を明確に設定してください（例：2024年12月1日から2025年1月31日）。
4.目標数値: 
  - プロモーションによって達成したい数値目標を具体的に設定してください（例：新規顧客獲得数100件、Webサイトアクセス数10,000PV、売上目標1,000万円）。
5.予算: 
  - プロモーションにかけられる予算を明確に設定してください（例：100万円）。
6.主要なプロモーション施策: 
  - 計画の骨子となる主要なプロモーション施策を決定してください（例：SNS広告、インフルエンサーマーケティング、クーポン配布）。

ステップ2: 具体的な行動計画を作成
ステップ1で設定した全体像に基づき、具体的な行動計画を作成してください。
1.各施策の詳細: 
 - ステップ1で決定した各プロモーション施策について、具体的な内容を詳細に記述してください。
 - 例：SNS広告
   - プラットフォーム: Instagram、Twitter
   - 広告フォーマット: 画像広告、動画広告
   - ターゲティング: 女性、20代、美容に関心あり
   - 予算配分: 各プラットフォームに50万円
   - 広告クリエイティブ: ターゲット層に響くデザインとメッセージ
   - 効果測定: 費用対効果（ROAS）、コンバージョン率
 - 例：インフルエンサーマーケティング
   - 対象インフルエンサー: 美容系、フォロワー数1万人以上
   - 契約内容: 投稿内容、投稿回数、報酬
   - 効果測定: エンゲージメント率、フォロワー増加数
 - 例：クーポン配布
   - クーポン内容: 10%オフ、送料無料
   - 配布方法: Webサイト、SNS、メールマガジン
   - 効果測定: クーポン利用率、売上増加
2.行動内容: 
 - 各施策を実行するために必要な具体的な行動内容を、担当者、期限、手順などを含めて明確に定義してください。
 - 例：SNS広告
   - 担当者: 広報担当
   - 期限: 2024年11月20日までに広告クリエイティブ作成
   - 手順: 広告アカウント作成 → ターゲティング設定 → 広告クリエイティブ設定 → 広告配信開始 → 効果測定
 - 例：インフルエンサーマーケティング
   - 担当者: マーケティング担当
   - 期限: 2024年11月15日までにインフルエンサー選定
   - 手順: インフルエンサー候補リスト作成 → 候補者への連絡 → 契約交渉 → 契約締結 → 投稿指示 → 効果測定
 - 例：クーポン配布
   - 担当者: 営業担当
   - 期限: 2024年11月25日までにクーポンコード発行
   - 手順: クーポンコード発行 → クーポン配布方法決定 → 告知文作成 → 配布開始 → 効果測定
3.スケジュール: 
 - 全体のスケジュールをガントチャートなどで可視化し、各施策の開始日と終了日を明確にしてください。
4.  KPI設定と効果測定: 
 - 各施策の成功を測るためのKPIを設定し、どのように効果測定を行うかを明確に記述してください。
 - 例：SNS広告：インプレッション数、クリック数、コンバージョン率
 - 例：インフルエンサーマーケティング：リーチ数、エンゲージメント率
 - 例：クーポン配布：利用率、売上への貢献度

ステップ3: 計画書の作成
上記ステップで検討した内容を、以下の構成でプロモーション計画書にまとめてください。
{プロモーション期間}に合わせた、{ターゲット顧客層}向けの、{目標数値}を伸ばすために、{使用する広告媒体}を活用し、具体的な{イベント企画内容}の計画と、具体的な行動内容を作成してください。

- プロモーション概要（目的、ターゲット顧客層、期間、予算、目標数値）
- 各プロモーション施策の詳細（内容、ターゲット、予算、KPI、スケジュール）
- 具体的な行動計画（担当者、期限、手順）
- 効果測定方法
- リスクと対策
- 緊急時対応プラン

[ルール]
- まず、ユーザーが抱える漠然としたプロモーション要求を、プロンプトに入力します。
  例えば、「観光客を増やしたい」「移住者を増やしたい」「地域産品をもっとPRしたい」といった抽象的な要望が考えられます。
- 次に、プロンプトは、入力された要求を分析し、具体的な目標設定を促します。
  例えば、「観光客数を年間10%増加させる」「5年間で100世帯の移住者を獲得する」「地域産品の売上を前年度比20%向上させる」といった具体的な目標を設定します。
- さらに、プロンプトは、目標達成のために必要な具体的な行動計画の立案を支援します。
  例えば、「ターゲット層に合わせた観光パンフレットを作成する」「移住希望者向け相談窓口を設置する」「地域産品を販売するオンラインショップを開設する」といった具体的な施策を検討します。
- また、プロンプトは、行動計画の実行に必要な予算や人員、スケジュールなどを考慮し、実現可能な戦略策定を支援します。
- 最後に、プロンプトは、効果測定方法を検討することで、プロモーション活動の成果を可視化します。
- 各ステップで得られた情報は、次のステップに活用してください。

[出力形式]
- 計画書はドキュメント形式で作成してください。
- 見出し、箇条書き、表、図などを効果的に使用し、読みやすく分かりやすい構成にしてください。
- 必要に応じて参考資料や補足資料を添付してください。
