# Prompt Aggregator (Local Version)

このフォルダには、Web上のプロンプト集をローカルPCで快適に閲覧・検索するためのアプリが含まれています。
インストール不要、インターネット接続なしでも（データ更新時以外）動作します。

## 使い方

### 1. アプリを開く
**`open_app.cmd`** をダブルクリックしてください。
ブラウザが立ち上がり、プロンプト集が表示されます。
- **検索**: キーワードで絞り込み
- **タグ**: カテゴリをクリックして絞り込み
- **詳細**: カードをクリックして全文表示＆コピー

### 2. データを更新する
新しいプロンプトを取得したい場合は、**`update_app.cmd`** をダブルクリックしてください。
- Webサイトから最新データを取得し、自動的に分類してアプリを更新します。
- ※インターネット接続が必要です。
- ※完了まで数分かかる場合があります。

## フォルダ構成
- `prompt-aggregator.html`: アプリ本体（これが全てです）
- `data/prompts.json`: プロンプトの生データ
- `scripts/`: 更新用プログラム（Python）

## 動作環境
- Windows PC
- Python 3.x がインストールされていること
